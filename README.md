# ☕ Java Fundamentals

Repositório dedicado ao estudo de Java, contendo exercícios e pequenos projetos organizados por temas, com foco no aprendizado prático dos principais conceitos da linguagem.

---

## 📚 Estrutura do Projeto

Os exercícios estão organizados por assunto:

```
src/
├── condicionais/
├── switchcase/
├── loops/
├── arrays/
├── matrizes/
├── listas/
├── set/
├── funcoes/
├── poo/
├── PrimeirosPassos.java
```

Cada pasta contém exemplos e exercícios relacionados ao tema, com classes independentes.

---

## 🧠 Conteúdos abordados

* Estruturas condicionais (`if`, `else`)
* Controle de fluxo com `switch`
* Laços de repetição (`for`, `while`, `do while`)
* Arrays e matrizes
* Listas e conjuntos (`List`, `Set`)
* Métodos e funções
* Programação Orientada a Objetos (POO)

---

## 🚀 Objetivo

Desenvolver uma base sólida em Java por meio de prática constante, evoluindo desde conceitos básicos até estruturas mais avançadas.

---

## ▶️ Como executar

1. Abra o projeto em uma IDE (IntelliJ, Eclipse, etc)
2. Navegue até a pasta `src`
3. Escolha um arquivo `.java`
4. Execute a classe (cada uma possui seu próprio `main`)

---

## 📌 Observações

* Cada arquivo representa um exercício independente
* O foco está na prática e na compreensão da lógica
* A organização segue a progressão do curso
